# Agent Remote Toolkit

把 AI Agent 从电脑里放出来 —— 远程指挥三件套方法论。

📖 **在线阅读**：https://lmrsama.github.io/agent-remote-toolkit/

🎯 **适合谁**：用 CodeBuddy / Cursor / Claude Code 等 agent 类 AI 编程工具的人。

---

## 三件套核心

1. **持久内存（Memory File System）** — 把对话记忆从 LLM context 挪到磁盘文件
2. **远程入口（Remote Channel）** — 让你从手机触达正在跑的 agent
3. **死命令护栏（Hard Constraints）** — 放开高权限模式后兜底防失控

---

## 配套 Skill

`skills/claw-remote-takeover/` — 一键启用远程化身模式的 CodeBuddy skill。

安装：把整个目录复制到 `~/.workbuddy/skills/` 下即可。
